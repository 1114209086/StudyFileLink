### VI编辑
1.  vi filename 编辑某个文件  
2.  i 进行编辑模式 （ 非编辑模式下 ）
3.  Esc 退出编辑模式 （编辑模式下 ）
4.  ：wq 保存关闭 （非编辑模式下 ）
### GIT

git stash pop	         本地改动显示出来
